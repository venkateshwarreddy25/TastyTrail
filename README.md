# CampusEats — Modern Canteen Ordering System

A production-ready Final Year CSE project built with React, Firebase, and Vertex AI.

## Quick Start

1. Install dependencies
```bash
npm install
cd functions && npm install && cd ..
```

2. Configuration
Copy `src/firebase.ts` and set up your Firebase project credentials. Ensure Firestore, Auth, Hosting, and Functions are enabled.

3. Run Development Server
```bash
npm start
```

For full setup explanation, architecture details, and Viva questions, please refer to the `DOCUMENTATION.md` file!
